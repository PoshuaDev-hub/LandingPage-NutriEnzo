# NutriEnzo — Landing Page Profesional

NutriEnzo es una Landing Page de alto rendimiento diseñada para una consultoría de nutrición clínica. El proyecto combina un diseño estético y moderno con una gestión dinámica de contenidos mediante Supabase.

## 🚀 Tecnologías Utilizadas

- **Vite**: Herramienta de construcción ultra rápida para el desarrollo frontend.
- **Vanilla JavaScript**: Lógica pura sin frameworks pesados para maximizar el rendimiento.
- **CSS3 (Custom Properties)**: Sistema de diseño basado en variables para una gestión de estilos limpia.
- **Supabase**: Backend-as-a-Service para la base de datos de noticias y el almacenamiento de imágenes.
- **FontAwesome & Google Fonts**: Tipografía e iconografía premium.

## ✨ Características Principales

- **Diseño Premium & Responsive**: Optimizado para dispositivos móviles y escritorio con animaciones suaves.
- **Dashboard Administrativo**: Panel oculto para gestionar las noticias en tiempo real.
- **Galería de Novedades Dinámica**: Carrusel conectado a la base de datos de Supabase con soporte para imágenes 3:2.
- **Sistema de Seguridad**: Acceso al panel administrativo protegido por clave secreta.
- **Integración de Agendamiento**: Modal interactivo para facilitar la conversión de pacientes.
